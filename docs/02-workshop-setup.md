# Workshop Setup

## Computer Setup

- Browser with Web MIDI support if using live preview.
- Ableton Live installed and tested.
- TouchDesigner project installed and tested.
- MIDI loopback device installed if required.

	- For MacOS native IAC Driver works well

- Local copies of template Ableton sets and TouchDesigner patches.
