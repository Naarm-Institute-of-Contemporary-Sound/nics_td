## Required Dependencies

### Blender (free)

Any blender version above 4.2 will work. This project was created with Blender 5.1.2 which can be downloaded [here](https://www.blender.org/download/)

You'll then need to follow the instructions to download and install the [Blender DMX Addon](https://blenderdmx.eu/) (it's an easy drag and drop install!)

Once complete, you can download and open up the [Blender project file]( https://drive.google.com/drive/u/3/folders/1sAy4BP1EEYKHOK9oG2kL2qmoLLUDh9_x)

### Touch Designer (free)

TouchDesigner (TD) can de downloaded [here](https://derivative.ca/download), the latest version should work best. (requires 2025.32820 or higher)

You can find the TouchDesigner NICS project file in this GitHub repo, which can be downloaded once off as a [zip file](https://github.com/Naarm-Institute-of-Contemporary-Sound/nics_td/archive/refs/heads/main.zip). Or if you're a Git enjoyer, it can be cloned with [ssh](git@github.com:Naarm-Institute-of-Contemporary-Sound/nics_td.git) or [https](https://github.com/Naarm-Institute-of-Contemporary-Sound/nics_td.git) (please feel free to push to a branch and make a PR)

The NICS TD project file relies on a few supporting files that come with it, so please make sure to keep it in it's current folder, moving it may break it!

### NICS MIDI Assignment Tool

This free online component can be accessed [here](https://naarm-institute-of-contemporary-sound.github.io/nics_midi_lighting_tool/)

This will help you to generate MIDI files that can be played back to Touch Designer and trigger lights! There are a few necessary MIDI CC channels that need to be set

### Ableton / a DAW

The lite version of Ableton will work fine here, it comes bundled with a lot of MIDI controllers for free, or can be purchased online for a few $ from sketchy websites. The full version of Ableton can be purchased from Ableton themselves. Alternatively there is a free trial downloadable [here](https://www.ableton.com/en/trial/) 

Other DAWs capable of outputting MIDI notes and MIDI CC should work fine here, but are not documented yet
