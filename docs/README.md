# NICS MIDI Lighting Tool Workshop Tutorial

This folder is a workshop documentation skeleton for teaching artists how to turn music into lighting control material, preview it in the browser, export MIDI, and present the result through the NICS Ableton, TouchDesigner, and DMX system.

## Workshop Aim

By the end of the workshop, participants should be able to:

- Explain the difference between offline analysis and online analysis.
- Convert audio or direct MIDI into the NICS low-note lighting trigger format.
- Shape how source pitch ranges map onto fixture groups.
- Preview the mapped MIDI against the song.
- Export a MIDI file that can be used in the NICS performance workflow.
- Understand how their choices affect what the audience sees in the room.
